Server is Up and Running

You see default page served by the server. You can change this page to whatever you need, simply by replacing index.html in your server folder.
Whatever you put in the server folder will be accessible through the URL. For example to access this site you are using certificate available at this TODO URL, mapped to your server's folder.
If you put file bigger than there is available memory and then request it via TODO URL, server will be stopped with an error.
